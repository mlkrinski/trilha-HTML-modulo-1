Surgimento da internet

< Voltar

A internet surgiu na década de 1960 nos EUA com a ARPANET, uma rede militar para comunicação descentralizada, e evoluiu para a rede global que conhecemos com a popularização do acesso comercial e a criação da World Wide Web nos anos 90, chegando ao Brasil no final dos anos 80, inicialmente em universidades, e abrindo para o público geral em 1995.

No Mundo (Origem Militar/Acadêmica):

  1. Anos 60 (Guerra Fria): O Departamento de Defesa dos EUA criou a ARPANET (Advanced Research Projects Agency Network) com o objetivo de ter uma rede de comunicação resistente a ataques, conectando instituições de pesquisa.
  2. 1969: A primeira conexão da ARPANET foi estabelecida, com a primeira mensagem enviada entre UCLA e Stanford.
  3. Anos 80: A tecnologia se expandiu para universidades e centros de pesquisa, com protocolos como o TCP/IP sendo desenvolvidos, fundamentais para a interconexão de redes.

Em resumo, a internet nasceu de uma necessidade pessoal militar, foi desenvolvida no ambiente acadêmico e, com a abertura comercial nos anos 90, tornou-se a ferramenta global de comunicação e informação que conhecemos.

"A Web não apenas conecta máquinas, conecta pessoas."

Tim Berners-Lee.

Como funciona o HTML

< Voltar

O HTML funciona como o esqueleto de uma página web, usando "tags" (como <p> para parágrafos ou <img> para imagens) para estruturar e organizar o conteúdo (textos, imagens, links). O navegador interpreta esse código, transformando as tags em elementos visuais para o usuário.

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web. Outras tecnologias além do HTML geralmente são usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento (JavaScript) de uma página da web.

"Hipertexto" refere-se aos links que conectam páginas da Web entre si, seja dentro de um único site ou entre sites. Links são um aspecto fundamental da web. Ao carregar conteúdo na Internet e vinculá-lo a páginas criadas por outras pessoas, você se torna um participante ativo na world wide web.

O HTML usa "Marcação" para anotar texto, imagem e outros conteúdos para exibição em um navegador da Web. A marcação HTML inclui "elementos" especiais, como <head>, <title>, <body>, <header>, <footer>, <article>, <section>, <<p>>, <div>, <span>, <img>, <audio>, <nav>, <video>, <ul>, <ol>, <li> e muitos outros.

Um elemento HTML é separado de outro texto em um documento por "tags", que consistem no nome do elemento entre "<" e ">". O nome de um elemento dentro de uma tag é insensível a maiúsculas e minúsculas. Isto é, pode ser escrito em maiúsculas, minúsculas ou um mistura.

O que são Clients

< Voltar

Clients se refere a dispositivos ou programas que iniciam solicitações por serviços ou dados sendo um navegador ou aplicativo no celular. O cliente é o ponto de interação do usuário, ou interface, que envia as requisições e recebe a resposta processada pelo servidor que é onde centraliza os recursos.

O que são Servers

< Voltar

Servers se refere ao computador ou programa centralizado que fornece recursos, dados ou serviços para múltiplos clientes que solicitam através de uma rede interna ou externa. O servidor compartilha essa resposta de forma eficiente e controlada de volta para o cliente.

Entendendo a diferença entre Lista Ordenadas e Não Ordenadas

< Voltar

A principal diferença é que listas ordenadas <ol> usam números ou letras (1, 2, 3 ou a, b, c) indicando sequência, como em passos ou rankings, enquando listas não ordenadas <ul> usam marcadores como bolinhas ou quadrados mostrando apenas uma coleção de itens onde a ordem não importa. Ambas são construidas com itens <il>.
